Hi. This is the first iteration of a Georgian morphological analysis... dictionary... thingy that I've made for chopping up Georgian verbs. All definitions come from the Rayfield Comprehensive Georgian-English Dictionary. Although the original purpose of this project was to create a resource for quickly analyzing Georgian verbs in the wild with minimum fuss, I'm hoping to add more entries for other parts of speech in the future. You can use the box below to look up a verb.

About: Currently this dictionary contains about 7,000 verb roots and 30,000 entries. When a verb is entered, the analyzer isolates the root and labels other morphology to the best of its ability, then returns a set of possible definitions from Rayfield's dictionary. Terminology is mostly borrowed from PJ Hillery's excellent overview of Georgian grammar, which can be found online here.

Capabilities: Georgian verb morphology can be complex, but if the root is in the dictionary it should be available to the analyzer, even if the form is irregular. For example, entering ვთქვით 'we said' or გიტყვი 'I will say to you' returns the same entry in spite of the irregularity in the roots. Agreement for person and number occasionally confuse the analyzer, but only occasionally.

Incapabilities: Currently there are no dictionary entries for verbal nouns (masdars) yet, although it will often return the correct root if you enter one. It does not, however, recognize when a form is ungrammatical. Non-existent forms like *გადავთარგმნის are treated the same as any other verb, although this could change fairly soon. The dictionary is very much a work in progress, and I'll be adding more entries as time goes by. Because the dictionary entries were all created automatically, there are occasional errors -- mislabelings, empty definitions, and typos from the original dictionary. Fortunately, most errors are systematic, so if you run across one there's a good chance that I can use it to correct a bunch of other ones once I know they exist. If you have any questions, comments or ideas, or find a bug, please get in touch with me at gflinn[at]uchicago[.]edu.

Future: 30,000 verbs is only scratching the surface of the Rayfield dictionary. This is the first iteration, but over the rest of the summer (it's 2017 now) the system will be undergoing plenty of refinement. Eventually I hope this can be a general resource for linguists working on Georgian -- with a dictionary of fully analyzed morphology, it won't be hard to lookup by morpheme type. If you want to, say, see a list of all of the verbs that have ი as a version vowel and ავ as a stem formant (there are currently 525), then the next step is to provide that.
